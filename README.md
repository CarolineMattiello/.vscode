# Title
Monthly Sales Tracker
# Discription
This program calculates the monthly sales a company or person will have. It predicts this by taking the weekly sales and multiplying it by 4.345 which is the average amount of weeks in a month.
# Run instructions
python app.py
# Sample output
Enter a week's amount of sales: 1000
You are on track to earn $17,380 this month.
